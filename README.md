# 🍽️ Manam Restaurant Website

✨ A premium Django-based restaurant web application for online food ordering and restaurant management.

---

## 🚀 Features

- 🍕 Online Food Ordering
- 📋 Dynamic Restaurant Menu
- 👤 User Authentication
- 💳 Razorpay Payment Integration
- 📱 Responsive UI Design
- 🛠️ Admin Dashboard
- 🖼️ Static Image Support
- ⚡ Fast Django Backend

---

## 🛠️ Technologies Used

| Technology | Usage |
|---|---|
| 🐍 Python | Backend |
| 🌐 Django | Web Framework |
| 🎨 HTML/CSS | Frontend |
| ⚡ JavaScript | UI Interaction |
| 🗄️ SQLite | Database |
| 💳 Razorpay | Payments |
| 🚀 Gunicorn | Deployment |
| ☁️ Render | Hosting |

---

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/sankum136/Manam.git
```

### Open Project

```bash
cd Manam
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Migrations

```bash
python manage.py migrate
```

### Start Server

```bash
python manage.py runserver
```
